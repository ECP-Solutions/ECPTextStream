# CloseStream
{: .fs-9 }

Closes the current text file stream.
{: .fs-6 .fw-300 }

---

## Syntax

*expression*.`CloseStream`

### Parameters

_None_

>⚠️**Caution**
>{: .text-grey-lt-000 .bg-green-000 }
>User will lose the connection to text file after close the current stream.
{: .text-grey-dk-300 .bg-yellow-000 }

### Return value

_None_
